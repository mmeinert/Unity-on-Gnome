# Unity-on-Gnome
unity style with gnome power

HOW TO INSTALL:

Just copy and paste this commands in terminal

```bash
wget https://raw.githubusercontent.com/RabbitHols/Unity-on-Gnome/master/installer.sh
chmod +x installer.sh 
./installer.sh
```
